A simple, better looking scrollbar for your website.

== Installation ==

1. Upload plugin to the /wp-content/plugins
2. Activate the plugin through the "Plugins" menu in WordPress
3. Enjoy

== Frequently Asked Questions ==

= Do I need to configure any settings? =
no, just activate the plugin, and it will add it's css.
